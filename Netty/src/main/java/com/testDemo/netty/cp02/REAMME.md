| |同步阻塞I/O(BIO)|伪异步I/O|非阻塞I/O(NIO)|异步I/O(AIO)|

|I/O类型(I/O)|同步I/O|同步I/O|同步I/O(I/O多路复用)|异步I/O|
|API使用难度|简单|简单|非常复杂|复杂|
|调试难度|简单|简单|复杂|复杂|
|可靠性|非常差|差|高|高|
|吞吐量|低|中|高|高|
